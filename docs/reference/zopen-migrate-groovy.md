<h1 align="center">ZOPEN-MIGRATE-GROOVY</h1>




<h2>NAME</h2>



<p style="margin-left:6%; margin-top: 1em">zopen-migrate-groovy
- manual page for zopen-migrate-groovy 0.8.5</p>

<h2>DESCRIPTION</h2>



<p style="margin-left:6%; margin-top: 1em">zopen-migrate-groovy
[--version|--help]</p>

<p style="margin-left:15%; margin-top: 1em">simple script
to create a cicd-dev.groovy and cicd-stable.groovy from a
cicd.groovy. By no means bullet-proof but might save some
typing Has no checks in it and requires you fill in the
...'s in the generated files.</p>

<p style="margin-left:15%; margin-top: 1em">Also, spacing
is a bit off - but easy enough to change post-processing in
the editor.</p>
