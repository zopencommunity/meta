NAME

zopen-query − manual page for zopen-query 0.8.4

SYNOPSIS

zopen-query [OPTION] [VERB] [PACKAGE]

DESCRIPTION

zopen−query − a utility for zopen community to query packages and repos.

VERB is the action to take, which is one of

−−list, −−remote−search, −−installed

PACKAGE is a package, specified for −−remote−search

Verbs:

−i, −−installed

list installed zopen community.

−−list

list all available zopen community.

−−remote−search

regex match package against available zopen community

OPTIONS

−d, −−details

include full details for listings.

−−category <category> apply a category filter.

−−help

print this help

−−no−header,

suppress the header for the output.

−−no−version,

suppress version information, return package names.

−v

run in verbose mode.

−−version

print version
