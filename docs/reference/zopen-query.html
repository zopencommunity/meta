NAME

zopen-query − manual page for zopen-query 0.8.0

SYNOPSIS

zopen-query [OPTION] [VERB] [PACKAGE]

DESCRIPTION

zopen−query − a utility for z/OS Open Tools to query packages and repos.

VERB is the action to take, which is one of

−−list, −−remote−search, −−installed

PACKAGE is a package, specified for −−remote−search

Verbs:

−−list

list all available z/OS Open Tools.

−−remote−search

regex match package against available z/OS Open Tools

−i, −−installed

list installed z/OS Open Tools.

OPTIONS

−v

run in verbose mode.

−d, −−details

include full details for listings.

−−no−header,

suppress the header for the output.

−−no−version,

suppress version information, return package names.

−−help

print this help

−−version

print version
