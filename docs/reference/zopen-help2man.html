NAME

zopen-help2man − manual page for zopen-help2man 0.8.4

SYNOPSIS

zopen-help2man [OPTION] DIRECTORY

DESCRIPTION

zopen−help2man − generate man pages from help and version information for zopen and zopen−* tools.
DIRECTORY

the existing directory to write the man pages.

OPTIONS

−−help

display this help and exit

−−version

print version

Example:"
zopen−help2man /tmp/man1

write the man pages to the /tmp/man1 directory.
