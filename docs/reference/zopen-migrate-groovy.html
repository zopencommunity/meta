NAME

zopen-migrate-groovy − manual page for zopen-migrate-groovy 0.8.4

DESCRIPTION

zopen−migrate−groovy [−−version|−−help]

simple script to create a cicd−dev.groovy and cicd−stable.groovy from a cicd.groovy. By no means bullet−proof but might save some typing Has no checks in it and requires you fill in the ...’s in the generated files.

Also, spacing is a bit off − but easy enough to change post−processing in the editor.
