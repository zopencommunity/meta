NAME

zopen-version − manual page for zopen-version 0.8.4

SYNOPSIS

zopen-version [OPTION] tool

DESCRIPTION

zopen−version prints version information for zopen tools.

OPTIONS

−−help

display this help and exit.

−−version

print version.
