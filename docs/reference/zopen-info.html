NAME

zopen-info − manual page for zopen-info 0.8.4

SYNOPSIS

zopen-info [OPTION] PACKAGE

DESCRIPTION

zopen−info − display detailed information about a zopen package.

PACKAGE is the name of a zopen package.

OPTIONS

−h, −−help

print this help

−v, −−verbose

run in verbose mode.

−r, −−remote

force a remote lookup

−−version

print version
