<h1 align="center">ZOPEN-REMOVE</h1>












<h2>NAME</h2>


<table><tr><td style="width: 25%; vertical-align: top;"><b>zopen-remove
- manual page for zopen-remove 0.8.5

<h2>SYNOPSIS</h2>



<b>zopen-remove</b>
[<i>OPTION</i>] [<i>PACKAGE</i>] ...

<h2>DESCRIPTION</h2>



zopen-remove
is a utility for zopen community to remove an installed
package or packages.

<h2>OPTIONS</h2>



<b>-h</b>,
<b>--help</b>, -?</b></td><td style="vertical-align: top;">display this help and exit.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-p</b>,
<b>--purge</b></td><td style="vertical-align: top;">remove package, the versioned
directory, and any cached files.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-v</b>,
<b>--verbose</b></td><td style="vertical-align: top;">run in verbose mode.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--version</b></td><td style="vertical-align: top;">print version.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-y</b>,
<b>--yes</b></td><td style="vertical-align: top;">automatically answer yes to
prompts.</td></tr></table>

<h2>EXAMPLES</h2>


<table><tr><td style="width: 25%; vertical-align: top;"><b>zopen remove foo
bar</b></td><td style="vertical-align: top;">interactively remove the foo
and bar packages if installed</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>zopen remove --yes
foo bar</b></td><td style="vertical-align: top;">remove the foo and bar packages
if installed, without asking for confirmation</td></tr></table>
