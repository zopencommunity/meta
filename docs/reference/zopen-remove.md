<h1 align="center">ZOPEN-REMOVE</h1>




<h2>NAME
<a name="NAME"></a>
</h2>


<p style="margin-left:11%; margin-top: 1em">zopen-remove
&minus; manual page for zopen-remove 0.8.4</p>

<h2>SYNOPSIS
<a name="SYNOPSIS"></a>
</h2>



<p style="margin-left:11%; margin-top: 1em"><b>zopen-remove</b>
[OPTION] [PACKAGE] ...</p>

<h2>DESCRIPTION
<a name="DESCRIPTION"></a>
</h2>



<p style="margin-left:11%; margin-top: 1em">zopen&minus;remove
is a utility for zopen community to remove an installed
package or packages.</p>

<h2>OPTIONS
<a name="OPTIONS"></a>
</h2>



<p style="margin-left:11%; margin-top: 1em"><b>&minus;h</b>,
<b>&minus;&minus;help</b>, &minus;?</p>

<p style="margin-left:22%;">display this help and exit.</p>

<p style="margin-left:11%;"><b>&minus;p</b>,
<b>&minus;&minus;purge</b></p>

<p style="margin-left:22%;">remove package, the versioned
directory, and any cached files.</p>

<p style="margin-left:11%;"><b>&minus;v</b>,
<b>&minus;&minus;verbose</b></p>

<p style="margin-left:22%;">run in verbose mode.</p>


<p style="margin-left:11%;"><b>&minus;&minus;version</b></p>

<p style="margin-left:22%;">print version.</p>

<p style="margin-left:11%;"><b>&minus;y</b>,
<b>&minus;&minus;yes</b></p>

<p style="margin-left:22%;">automatically answer yes to
prompts.</p>

<h2>EXAMPLES
<a name="EXAMPLES"></a>
</h2>


<p style="margin-left:11%; margin-top: 1em">zopen remove
foo bar</p>

<p style="margin-left:22%;">interactively remove the foo
and bar packages if installed</p>

<p style="margin-left:11%;">zopen remove &minus;&minus;yes
foo bar</p>

<p style="margin-left:22%;">remove the foo and bar packages
if installed, without asking for confirmation</p>
