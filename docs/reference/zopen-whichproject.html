NAME

zopen-whichproject − manual page for zopen-whichproject 0.8.4

SYNOPSIS

zopen-whichproject [OPTION]... <COMMAND|LIBRARY>

DESCRIPTION

zopen−whichproject − Find which zopen project provides a command or library.

COMMAND is the name of an executable (e.g., ’git’, ’curl’). LIBRARY is the name of a library file (e.g., ’libssl.so’, ’libz.a’).

OPTIONS

−h, −−help

Print this help message.

−v, −−verbose

Run in verbose mode (print debug messages).

−−version

Print version information (requires zopen−version).
