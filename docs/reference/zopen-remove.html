NAME

zopen-remove − manual page for zopen-remove 0.8.4

SYNOPSIS

zopen-remove [OPTION] [PACKAGE] ...

DESCRIPTION

zopen−remove is a utility for zopen community to remove an installed package or packages.

OPTIONS

−h, −−help, −?

display this help and exit.

−p, −−purge

remove package, the versioned directory, and any cached files.

−v, −−verbose

run in verbose mode.

−−version

print version.

−y, −−yes

automatically answer yes to prompts.

EXAMPLES

zopen remove foo bar

interactively remove the foo and bar packages if installed

zopen remove −−yes foo bar

remove the foo and bar packages if installed, without asking for confirmation
