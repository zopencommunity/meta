NAME





zopen-split-patch
- manual page for zopen-split-patch 0.8.4

SYNOPSIS





**zopen-split-patch**
[*OPTION*] [*PATCHFILE*]

DESCRIPTION





zopen-split-patch
- split a patch file into one patch file for each
patch.

Patches are
written to split-patches in the current directory.

OPTIONS




<table width="100%" border="0" rules="none" frame="void"
       cellspacing="0" cellpadding="0">





**--help**




display this help and exit.





**--version**

print version.


**Example:**


zopen-split-patch P1.patch

write P1.patch into multiple
patches under split-patches
