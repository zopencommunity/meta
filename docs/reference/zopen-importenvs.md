<h1 align="center">ZOPEN-IMPORTENVS</h1>

<h2>NAME</h2>

<p style="margin-left:11%; margin-top: 1em">zopen-importenvs
&minus; manual page for zopen-importenvs 0.8.0</p>

<h2 id="DESCRIPTION">DESCRIPTION</h2>

<p style="margin-left:11%; margin-top: 1em">USAGE: ..&/zopen-importenvs [path to buildenv to fetch dependency] If no argument is specified, zopen-importenvs will source all .env files from &/var&/lib&/jenkins&/zopen&/prod and &/var&/lib&/jenkins&/zopen&/boot. If an argument is specified and it represents a buildenv file, zopen-importenvs will source the ZOPEN_*_DEPS specified dependencies from &/var&/lib&/jenkins&/zopen&/prod and &/var&/lib&/jenkins&/zopen&/boot</p>
