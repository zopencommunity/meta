<h1 align="center">ZOPEN-HELP2MAN</h1>










<h2>NAME</h2>


<table><tr><td style="width: 25%; vertical-align: top;"><b>zopen-help2man
- manual page for zopen-help2man 0.8.5

<h2>SYNOPSIS</h2>



<b>zopen-help2man</b>
[<i>OPTION</i>] <i>DIRECTORY</i>

<h2>DESCRIPTION</h2>



zopen-help2man
- generate man pages from help and version information
for zopen and zopen-* tools. <br>
DIRECTORY</b></td><td style="vertical-align: top;">the existing directory to write
the man pages.</td></tr></table>

<h2>OPTIONS</h2>


<table>
<tr><td style="width: 25%; vertical-align: top;"><b>--help</b></td><td style="vertical-align: top;">display this help and exit</td></tr>
<tr><td style="width: 25%; vertical-align: top;"><b>--version</b></td><td style="vertical-align: top;">print version</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>Example:&quot;
<br>
zopen-help2man <i>/tmp/man1</i></b></td><td style="vertical-align: top;">write the man pages to the
<i>/tmp/man1</i> directory.</td></tr></table>
