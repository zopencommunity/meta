<h1 align="center">ZOPEN-HELP2MAN</h1>




<h2>NAME</h2>


<p style="margin-left:6%; margin-top: 1em">zopen-help2man -
manual page for zopen-help2man 0.8.5</p>

<h2>SYNOPSIS</h2>



<p style="margin-left:6%; margin-top: 1em">zopen-help2man
[OPTION] DIRECTORY</p>

<h2>DESCRIPTION</h2>


<p style="margin-left:6%; margin-top: 1em">zopen-help2man -
generate man pages from help and version information for
DIRECTORY</p>

<p style="margin-left:15%;">the existing directory to write
the man pages.</p>

<h2>OPTIONS</h2>



<p style="margin-left:6%; margin-top: 1em">--help</p>

<p style="margin-left:15%;">display this help and exit</p>

<p style="margin-left:6%;">--version</p>

<p style="margin-left:15%;">print version</p>

<p style="margin-left:6%; margin-top: 1em">Example:&quot;
zopen-help2man /tmp/man1</p>

<p style="margin-left:15%;">write the man pages to the
/tmp/man1 directory.</p>
