<h2>NAME</h2>



<table><tr><td style="width: 25%; vertical-align: top;"><b>zopen-whichproject

<h2>SYNOPSIS</h2>



<b>zopen-whichproject</b>
[<i>OPTION</i>]... <i><code>&lt;COMMAND|LIBRARY&gt;</code></i>

<h2>DESCRIPTION</h2>



zopen-whichproject
- Find which zopen project provides a command or
library.</b></td><td style="vertical-align: top;">COMMAND is the
name of an executable (e.g., 'git',
'curl'). LIBRARY is the name of a library file
