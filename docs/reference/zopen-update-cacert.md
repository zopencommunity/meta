<h1 align="center">ZOPEN-UPDATE-CACERT</h1>











<h2>NAME</h2>



<table><tr><td style="width: 25%; vertical-align: top;"><b>zopen-update-cacert
- manual page for zopen-update-cacert 0.8.5

<h2>DESCRIPTION</h2>



zopen-update-cacert:
Update your cacert.pem file to the latest CA certificates
extracted from Mozilla.

<h3>Syntax:</h3>



zopen-update-cacert
[-fhv] [<code>&lt;directory&gt;</code>]

<h2>OPTIONS</h2>



<b>-f</b>,
<b>--force</b></td><td style="vertical-align: top;">update cacert.pem file even if
up to date.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-h</b>,
<b>--help</b></td><td style="vertical-align: top;">print this help.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-i</b>,
<b>--insecure-fallback</b></td><td style="vertical-align: top;">(UNSAFE) Skip certificate
validation during download.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-v</b>,
<b>--verbose</b></td><td style="vertical-align: top;">print verbose messages.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--version</b></td><td style="vertical-align: top;">print version.</td></tr></table>

<h3>Parameters:</h3>



<p style="margin-left:18%; margin-top: 1em"><code>&lt;directory&gt;</code>:
the directory where the cacert.pem will be updated.</p>

<p style="margin-left:18%; margin-top: 1em">The default
directory location is:
<i>/u/tejas/new_zopen/zostools_dev/doc/meta</i></p>
