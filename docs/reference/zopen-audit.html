NAME

zopen-audit − manual page for zopen-audit 0.8.2

SYNOPSIS

zopen-audit [OPTION]

DESCRIPTION

zopen−audit is a utility for z/OS Open Tools to check for vulnerabilities in your installed packages.

OPTIONS

−v, −−verbose

run in verbose mode.

−−version

print version.

EXAMPLES

zopen audit

check for vulnerabilities in all installed packages
