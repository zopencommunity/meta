NAME




zopen-audit
- manual page for zopen-audit 0.8.4

SYNOPSIS





**zopen-audit**
[*OPTION*]

DESCRIPTION





zopen-audit
is a utility for zopen community to check for
vulnerabilities in your installed packages.

OPTIONS





**-v**,
**--verbose**

run in verbose mode.


**--version**

print version.

**-u**,
**--update**,
**--upgrade**

attempt to resolve
vulnerabilities by upgrading packages.

**-r**,
**--remove**

remove packages with
vulnerabilities.

EXAMPLES




zopen audit

check for vulnerabilities in
all installed packages
