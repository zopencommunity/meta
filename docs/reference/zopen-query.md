NAME




zopen-query
- manual page for zopen-query 0.8.4

SYNOPSIS





**zopen-query**
[*OPTION*] [*VERB*] [*PACKAGE*]

DESCRIPTION





zopen-query
- a utility for zopen community to query packages and
repos.

VERB is the
action to take, which is one of

<table width="100%" border="0" rules="none" frame="void"
       cellspacing="0" cellpadding="0">





**--list**,
**--remote-search**,
**--installed** 




PACKAGE is a
package, specified for
**--remote-search**

**Verbs:**

**-i**,
**--installed**

list installed zopen
community.

<table width="100%" border="0" rules="none" frame="void"
       cellspacing="0" cellpadding="0">





**--list**




list all available zopen community.





**--remote-search**

regex match package against
available zopen community

OPTIONS





**-d**,
**--details**

include full details for
listings.

<table width="100%" border="0" rules="none" frame="void"
       cellspacing="0" cellpadding="0">





**--category**
&lt;category&gt; apply a category filter.




**--help**

print this help


**--no-header**,

suppress the header for the
output.


**--no-version**,

suppress version information,
return package names.

<table width="100%" border="0" rules="none" frame="void"
       cellspacing="0" cellpadding="0">





**-v**




run in verbose mode.





**--version**

print version
