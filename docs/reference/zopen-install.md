<h1 align="center">ZOPEN-INSTALL</h1>




<h2>NAME</h2>


<p style="margin-left:6%; margin-top: 1em">zopen-install -
manual page for zopen-install 0.8.5</p>

<h2>SYNOPSIS</h2>



<p style="margin-left:6%; margin-top: 1em">zopen-install
[OPION] [PACKAGE]</p>

<h2>DESCRIPTION</h2>


<p style="margin-left:6%; margin-top: 1em">zopen-install is
a utility to download/install a zopen community package.</p>

<p style="margin-left:15%; margin-top: 1em">[PACKAGE] is a
package to install. Multiple packages can be specified.</p>

<h2>OPTIONS</h2>



<table width="100%" border="0" rules="none" frame="void"
       cellspacing="0" cellpadding="0">

<tr valign="top" align="left">

<td width="6%"></td>


<td width="7%">


<p style="margin-top: 1em">--all</p></td>


<td width="2%"></td>


<td width="58%">


<p style="margin-top: 1em">download/install all zopen
community packages.</p></td>


<td width="27%">
</td>
</tr>

</table>


<p style="margin-left:6%;">--cache-only</p>

<p style="margin-left:15%;">do not install
dependencies.</p>

<p style="margin-left:6%;">--download-only</p>

<p style="margin-left:15%;">download package to current
directory.</p>

<p style="margin-left:6%;">--help</p>

<p style="margin-left:15%;">print this help.</p>

<p style="margin-left:6%;">--install-or-upgrade</p>

<p style="margin-left:15%;">installs the package if not
installed, or upgrades the package if installed.</p>


<p style="margin-left:6%;">--bypass-prereq-checks</p>

<p style="margin-left:15%;">Ignores pre-req checks</p>

<p style="margin-left:6%;">--local-install</p>

<p style="margin-left:15%;">download and unpackage to
current directory.</p>

<p style="margin-left:6%;">--no-deps</p>

<p style="margin-left:15%;">do not install
dependencies.</p>

<p style="margin-left:6%;">--no-set-active</p>

<p style="margin-left:15%;">do not change the pinned
version.</p>

<p style="margin-left:6%;">--nosymlink</p>

<p style="margin-left:15%;">do not integrate into
filesystem through symlink redirection.</p>

<p style="margin-left:6%;">-r,
--reinstall</p>

<p style="margin-left:15%;">reinstall already installed
zopen community packages.</p>


<table width="100%" border="0" rules="none" frame="void"
       cellspacing="0" cellpadding="0">

<tr valign="top" align="left">

<td width="6%"></td>


<td width="78%">


<p style="margin-top: 1em">--release-line [stable,
dev] the release line to build off of.</p></td>


<td width="16%">
</td>
</tr>

</table>


<p style="margin-left:6%;">--select</p>


<p style="margin-left:15%;">select a version to
install.</p>

<p style="margin-left:6%;">--skip-upgrade</p>

<p style="margin-left:15%;">do not upgrade.</p>

<p style="margin-left:6%;">--force</p>

<p style="margin-left:15%;">force install, bypassing
locks.</p>

<p style="margin-left:6%;">-u, --update,
--upgrade</p>

<p style="margin-left:15%;">updates installed zopen
community packages.</p>

<p style="margin-left:6%;">-v, --verbose</p>

<p style="margin-left:15%;">print verbose messages.</p>

<p style="margin-left:6%;">--version</p>

<p style="margin-left:15%;">print version.</p>

<p style="margin-left:6%;">-y, --yes</p>

<p style="margin-left:15%;">automatically answer yes to
prompts.</p>
