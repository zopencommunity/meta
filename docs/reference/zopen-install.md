<h1 align="center">ZOPEN-INSTALL</h1>










<h2>NAME</h2>


<table><tr><td style="width: 25%; vertical-align: top;"><b>zopen-install
- manual page for zopen-install 0.8.5

<h2>SYNOPSIS</h2>



<b>zopen-install</b>
[<i>OPION</i>] [<i>PACKAGE</i>]

<h2>DESCRIPTION</h2>



zopen-install
is a utility to download/install a zopen community
package.</b></td><td style="vertical-align: top;">[PACKAGE] is a
package to install. Multiple packages can be specified.</td></tr></table>

<h2>OPTIONS</h2>


<table>
<tr><td style="width: 25%; vertical-align: top;"><b>--all</b></td><td style="vertical-align: top;">download/install all zopen
community packages.</td></tr>
<tr><td style="width: 25%; vertical-align: top;"><b>--cache-only</b></td><td style="vertical-align: top;">do not install
dependencies.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--download-only</b></td><td style="vertical-align: top;">download package to current
directory.</td></tr>
<tr><td style="width: 25%; vertical-align: top;"><b>--help</b></td><td style="vertical-align: top;">print this help.</td></tr>
<tr><td style="width: 25%; vertical-align: top;"><b>--install-or-upgrade</b></td><td style="vertical-align: top;">installs the package if not
installed, or upgrades the package if installed.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--bypass-prereq-checks</b></td><td style="vertical-align: top;">Ignores pre-req
checks</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--local-install</b></td><td style="vertical-align: top;">download and unpackage to
current directory.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--no-deps</b></td><td style="vertical-align: top;">do not install
dependencies.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--no-set-active</b></td><td style="vertical-align: top;">do not change the pinned
version.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--nosymlink</b></td><td style="vertical-align: top;">do not integrate into
filesystem through symlink redirection.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-r</b>,
<b>--reinstall</b></td><td style="vertical-align: top;">reinstall already installed
zopen community packages.</td></tr>
<tr valign="top" align="left">
<td width="9%"></td>
<td width="79%"><b>--release-line</b>
[stable, dev] the release line to build off of.</td>
<td width="12%">
</td></tr>
</table>

<p><b>--select</b></p></table>

<p style="margin-left:18%;">select a version to
install.</p>


<table><tr><td style="width: 25%; vertical-align: top;"><b>--skip-upgrade</b></td><td style="vertical-align: top;">do not upgrade.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--force</b></td><td style="vertical-align: top;">force install, bypassing
locks.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-u</b>,
<b>--update</b>,
<b>--upgrade</b></td><td style="vertical-align: top;">updates installed zopen
community packages.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-v</b>,
<b>--verbose</b></td><td style="vertical-align: top;">print verbose messages.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>--version</b></td><td style="vertical-align: top;">print version.</td></tr><tr><td style="width: 25%; vertical-align: top;"><b>-y</b>,
<b>--yes</b></td><td style="vertical-align: top;">automatically answer yes to
prompts.</td></tr></table>
